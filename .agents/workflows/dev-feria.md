## Pasos para iniciar el entorno de desarrollo

1. Verificar que las dependencias esten instaladas

```bash
npm list --depth=0
```

2. Iniciar el servidor de desarrollo en puerto 3100

```bash
npm run dev
```

3. Verificar configuracion de Supabase

```bash
echo "Verificando variables de entorno de Supabase..."
```

4. Mostrar URL de acceso local

```bash
echo "Servidor corriendo en http://localhost:3100"
```
